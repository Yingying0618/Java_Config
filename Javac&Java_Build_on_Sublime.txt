# Select Tool > Build_system > Build_new_system in sublime txt3
## The sublime txt3 will automatically pop-up a file for you


## //Paste following codes in your file

    ```
    "cmd": ["javac $file_name && java $file_base_name"],
    "file_regex": "^[ ]*File \"(...*?)\", line ([0-9]*)",
    "path": "/usr/bin/",
    "selector": "source.java",
    "shell": true
    ```


## //Save as "Java&JavaC.sublime-build", which saved at /Users/Which_User/Library/Application Support/Sublime Text 3/Packages


## //Quit sublime txt3, and open one JAVA files with sublime txt3
